## Project setup
```
composer install
```

## Project setup Migrations
```
php artisan migrate
```

## Project setup Login Screen
```
npm install
```

### Compiles and hot-reloads for development
```
php artisan run serve
```

### Run Login Screen
```
npm run dev
```
